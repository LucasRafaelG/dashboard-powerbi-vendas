# dashboard-powerbi-vendas

Descrição

Este projeto tem como objetivo o desenvolvimento de um dashboard interativo para análise de indicadores de negócio, com foco em dados de vendas. A proposta é facilitar a visualização de métricas relevantes e apoiar a tomada de decisão por meio de uma interface clara e objetiva.

Tecnologias Utilizadas
Power BI
Excel / CSV
Objetivos do Projeto
Construir um dashboard com indicadores-chave de desempenho (KPIs)
Analisar métricas como faturamento, volume de vendas e crescimento ao longo do tempo
Permitir a exploração dos dados por meio de filtros interativos
Organizar e modelar os dados para melhorar a performance das análises
Estrutura do Projeto
data/: base de dados utilizada para construção do dashboard
dashboard/: arquivo do Power BI (.pbix)
images/: capturas de tela do dashboard
